# PrimeMicroservicios-GenPrimos
Sistema distribuido de microservicios para generación de números primos con arquitectura escalable diseñado para generar números primos de cualquier tamaño (12+ dígitos) , procesamiento asíncrono (RabbitMQ), escalabilidad automática con reps en Kubernetes, persistencia en MySQL, migración automática y monitoreo en tiempo real de stats y progreso.
